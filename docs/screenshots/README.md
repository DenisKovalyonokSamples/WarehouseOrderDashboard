# Screenshots

Store UI and API screenshots for the main `README.md` references.

Expected files:

- `orders-tab.png`
- `stock-tab.png`
- `picking-tasks-tab.png`
- `dashboard-tab.png`
- `swagger.png`

Recommended capture tips:

1. Start both `Warehouse.Api` and `Warehouse.Wpf`.
2. Use a consistent window size for all WPF screenshots.
3. Capture each tab after data is loaded.
4. Capture Swagger with endpoint list visible.
5. Keep image width consistent (for example 1600px) for clean GitHub rendering.

Once files are added with these names, links in the root `README.md` will render automatically.
